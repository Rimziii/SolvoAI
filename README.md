# 🚀 SolvoAI – AI-Powered Code Debugging Assistant

SolvoAI is a full-stack web application that helps developers identify code errors, understand bug causes, and receive AI-powered explanations through an interactive chatbot interface.

## 🌟 Features

* 🤖 AI-powered debugging assistance
* 💬 Interactive chatbot for code-related queries
* 🔐 Secure user authentication with JWT
* 📜 Chat history management
* 🐛 Multi-language code analysis (C, C++, Python, JavaScript)
* 🎨 Modern and responsive UI
* ☁️ Live deployment with cloud hosting

## 🛠️ Technologies Used

### Frontend

* HTML5
* CSS3
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* SQLite

### AI Integration

* Groq API (LLM-powered responses)

### Security

* JWT Authentication
* bcrypt.js Password Hashing

## ⚡ Workflow

1. User submits code or a programming query.
2. Backend validates the request.
3. AI engine analyzes the input.
4. Response is generated with explanations and suggestions.
5. Results are displayed in a user-friendly interface.
